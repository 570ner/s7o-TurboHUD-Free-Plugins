# s7o Plugins for TurboHUD Free (THF)

TurboHUD Free download link:
https://turbohud.razorfish.dev/p/turbohud-free.html

## Install

1. Install TurboHUD Free from the link above.
2. Extract the plugin .ZIP into your TurboHUD Free folder so the files are here:

```text
../plugins/s7o/
```

Example:

```text
C:\TurboHUD\plugins\s7o\s7o_HUD_MENU.cs
```

3. Start TurboHUD Free.
4. Press **F8** in-game to open the **s7o HUD Menu**.

## Main menu

Default hotkey: **F8**

Use the HUD Menu to toggle plugins, change visual options, manage TTS alerts, set hotkeys, and control the built-in QoL tools.

## Notes

* This plugin pack is built for **TurboHUD Free**.
* Most options can be toggled or adjusted from the **F8 HUD Menu**.
* Input-automation helpers should be tested with your own keybinds and build setup.
* Some plugins were created, borrowed, ported, or modified to work together with the HUD Menu.
* If you find any bug, please report it.
